# MD to Platform — 功能与配置速览

> 本文独立于 README，便于快速对照「能做什么、要填什么」。安装与开发构建仍以根目录 `README.md` 为准。

---

## 一、环境与前提

| 项 | 说明 |
|----|------|
| 运行环境 | **Obsidian 桌面端**（插件标记为仅桌面） |
| 网络 | 需能访问你所填的 **大模型 / 生图 API**；公众号与视频号需能访问 **`api.weixin.qq.com`** |
| 微信 | **公众号**：需开发者 **AppID、AppSecret**；若后台启用 **IP 白名单**，须把当前上网出口 IP 加入 |
| 规则文件 | 插件目录下 `rules/` 至少含 `公众号扩写规则.md`、`gzh_to_xhs.md`、`wechat-markdown-rules.zh.md`；可选 `baoyu_xhs_images.md` 等 |

---

## 二、功能一览（按能力）

| 能力 | 作用简述 | 典型入口 |
|------|----------|----------|
| **扩写** | 读当前笔记为框架 → 生成 `publish_gzh.md`、`xhs_content.md`、`publish_xhs.md`（不改笔记正文） | 命令 `MDTP：扩写`、Ribbon、状态栏 **MDTP ▾** |
| **公众号草稿** | 读 `publish_gzh`，解析 `[配图：…]` 生图 → 花生风 HTML → 上传素材 → **微信公众平台草稿箱** | `MDTP：发布公众号草稿` |
| **小红书卡片图** | 读 `xhs_content.md`，按分隔符拆卡 → 导出 PNG；可选内置 `scripts/publish_xhs_redbook.py`（Python+xhs）或自定义命令 | `MDTP：发布小红书`、`MDTP：仅渲染小红书卡片图` |
| **公众号 → 小红书图文** | 长文/ `publish_gzh` → 生成小红书两份 md + 卡片 PNG | `MDTP：公众号文→小红书图文` |
| **Baoyu 风配图** | 长文拆段 → 多段生图提示词 → 逐张出图（与 HTML 卡片不同管线） | `MDTP：Baoyu 风配图` |
| **Seedance 视频任务** | 向 **火山方舟** 提交视频生成任务 | `MDTP：提交 Seedance 视频任务` |
| **视频号分片上传** | 库内 `.mp4` 经 **init + chunk** 上传（与公众号 **共用 AppID/Secret**） | `MDTP：视频号视频分片上传`、Ribbon **视频号** |

扩写可选配置包括 **公众号目标字数**（默认约 2000 汉字，与 `公众号扩写规则.md` 等一起约束模型）。

## 三、设置里要配什么（分组）

### 1. 文本大模型（扩写、文→卡、Baoyu 提示词等）

- **提供商**：DeepSeek / 智谱 / OpenAI 兼容（兼容需填 Base URL）
- **API Key**、**文本模型**（如 `deepseek-chat`）

### 2. 插图（公众号正文、Baoyu 等）

- **插图渠道**：**智谱**（如 `glm-image` + 尺寸）或 **火山方舟**（Key、接入域名、生图模型、尺寸、水印等）
- 可单独填 **插图专用 Key**

### 3. 微信公众平台（草稿 + 封面 + 视频号上传共用）

| 设置项 | 用途 |
|--------|------|
| **AppID / AppSecret** | 获取 `access_token`、上传正文图/封面素材、**创建草稿**；**视频号分片上传**也使用同一组凭证 |
| **排版主题** | 花生编辑器同源 HTML 内联样式 |
| **封面** | 纯色标题卡（背景预设、标题字号）或 **首张正文插图** |
| **视频号 · 分片上传** | init/chunk **路径**（相对微信域名）、**分片大小**、`media_type` 等，与官方文档一致 |

**注意**：IP 白名单、HTTPS、域名等以公众平台后台为准；插件不内置 HTTP 代理。

### 4. 火山方舟（仅当你用方舟插图或 Seedance）

- **方舟 API Key**、**接入域名**
- **生图模型 ID**、**Seedance 模型 ID**（视频任务命令用）

### 5. 小红书

- 卡片 **分隔正则**、**主题**、宽高、**DPR**、动态高度上限
- **卡片 PNG 保存位置**（工作流开启时目录由工作流固定）
- **无自定义命令时使用内置发布脚本**（默认开）+ **外部发布命令**（非空则覆盖内置）+ **Dry-run**；环境变量含 `MDT_PUBLISH_XHS`、`MDT_XHS_IMAGES_DIR`、`MDT_XHS_COOKIE`、`MDT_XHS_AS_PRIVATE` 等

### 6. 扩写与规则

| 设置项 | 说明 |
|--------|------|
| **公众号扩写目标字数** | 写入扩写请求，与 `公众号扩写规则.md` 一起约束篇幅 |
| **规则目录覆盖** | 留空 = 插件内 `rules/`；可填绝对路径或**库内相对路径** |

### 7. 写作目录工作流（Sandbox / Published）

- **工作流根**、**Sandbox 文件夹名**、**Published 文件夹名**
- 开启后：扩写 tmp 进 Sandbox，终稿进 Published 下 `gzh` / `xhs`；关闭则多写入**当前笔记同目录**

### 8. 其它

- **控制台调试日志**：排查按钮、路径、规则加载等
- **编辑区 MDTP 提示条**：左下角说明（可关）

## 五、界面入口（无需记全）

- **状态栏左侧**：**MDTP ▾** → 菜单（扩写、公众号、小红书、文→卡、葆玉图）
- **左侧 Ribbon**：同上 + **视频号**（打开上传对话框）
- **编辑器工具栏**：扩写、公众号、小红书、文→卡、葆玉图（无视频号时用命令）

---

## 六、和「只发小红书」的关系

插件 **不连接小红书官方发笔记接口**。小红书侧：生成 **文案 md + 卡片 PNG** 后，需在 **手机 App** 里发图发文，或自备 **本机脚本** 接自动化（插件只负责调起命令与环境变量）。

---

## 七、常见问题（一行）

| 现象 | 方向 |
|------|------|
| 草稿创建失败 | AppID/Secret、IP 白名单、网络 |
| 插图失败 | 智谱/方舟 Key、模型、额度 |
| 规则找不到 | `rules/` 路径或「规则目录覆盖」 |
| 视频号上传失败 | 路径与公众号是否同一开放平台账号、分片参数是否按文档 |
